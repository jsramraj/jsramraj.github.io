CSV Viewer & Editor

Browser-based CSV viewer and editing utility

A free, privacy-first CSV viewer and editor that runs entirely in your browser. Open, view, and edit CSV files instantly—no uploads, no backend, no ads—built for speed even with large files..

View full summary

Why this tool: It is free to use, has no ads, no tracking, and is fully open source. The app is lightweight, runs completely in the browser without any backend, and is performance tested with 10,000+ lines and CSV files larger than 2 MB.

Features: Open files from your device, drag-and-drop import, paste CSV from clipboard, and automatic delimiter detection. It also shows file metadata like rows, columns, file size, delimiter, and encoding, and supports column visibility toggle, drag-and-drop column reorder, column resize, freeze header row, freeze first column, and virtualized row rendering for smooth large-table viewing.

Web Tool · CSV · Data Utility · Open Source · Privacy First
